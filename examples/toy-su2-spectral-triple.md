# Toy $SU(2)$ Spectral-Triple Sketch

This is a sketch of a possible toy example. It is not yet a completed spectral triple.

## Algebra

Use smooth or truncated functions on $SU(2)$, or a finite matrix approximation to those functions.

Possible first choices:

- a small algebra generated by coordinate functions on $SU(2)$,
- a finite Peter-Weyl truncation,
- a matrix algebra carrying a clear $SU(2)$ action.

## Hilbert Space

Use spinor-valued $L^2(SU(2))$ as the conceptual model, or a finite Peter-Weyl truncation as the first computable model.

The finite model should state exactly what has been truncated and what structure remains.

## Operator

Use an $SU(2)$-Dirac-like operator or a Casimir-derived spectral operator.

The operator should come with:

- a domain, if infinite-dimensional,
- an explicit matrix form, if finite-dimensional,
- a spectrum or finite spectral ladder,
- commutator calculations with the chosen algebra.

## Goal

The goal is to define a small example that resembles a spectral triple but does not pretend to be fully proven yet.

The first useful deliverable is a technically inspectable object:

$$
(\mathcal A,\mathcal H,D)
$$

with explicit representation maps, operator formulas, and a list of which spectral-triple requirements are satisfied or remain open.
